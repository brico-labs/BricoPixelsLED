# BricoPixels LED

PCB diseñanda por BricoLabs para utilizar en proyectos de e-textile. Se utiliza un LED RGB [WS2812B](assets/WS2812B.pdf) que incorpora un circuito de control dentro de un componente de encapsulado 5050.
